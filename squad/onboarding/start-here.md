# Start Here

Five files. Here's which one you need.

---

## Getting started for the first time

**Step 1 — Read the setup guide**
Open `getting-started.md` — it walks you through setting up Superteam in Claude Desktop or claude.ai. Takes about 5 minutes.

**Step 2 — Paste the squad instructions into Claude**
Open `paste-into-claude.md` — copy the entire file and paste it into your Claude Project's instruction field. This is what activates the squad.

Once you've done both: start a conversation and say *"Superman — I'm [your name]. Let's start."*

---

## Starting a new project

Run through `new-project-checklist.md` with Superman. He walks the team through it — no agent starts working until it's complete.

---

## Introducing Superteam to colleagues

Open `superteam-intro.html` in any browser. It's an 8-slide presentation — use arrow keys, spacebar or the on-screen buttons to navigate. Press F for fullscreen.
