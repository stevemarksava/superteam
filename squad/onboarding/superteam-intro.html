How to Start

Path 1
Claude Desktop
or claude.ai
Easiest — no install

Create a new Project, paste the squad instructions, start talking. Works in any browser.

New Project
→ Paste squad instructions
→ Start talking
Path 2
Claude Code CLI
For developers

Clone the squad files into ~/.claude. Reload. Superman answers from your terminal or any IDE.

git clone ~/.claude
→ Reload session
→ Superman answers
Path 3
GitHub Copilot
in VS Code
In any repo

Copy the Copilot instructions file into any repo. The squad activates inside VS Code Chat.

Copy copilot-instructions.md
→ Into .github/ in your repo
→ Done

All three: just say an agent's name to activate them.  "Bob — blueprint this"

Using the Squad

You speak to Superman by default.

Address any agent by name to switch. They adopt that role for the response, then return to Superman on your next message.

Bob "Blueprint the data model for this Azure pipeline"
Kryptonite "Attack this plan before we build it"
Vera "What test cases are we missing here?"
Athanasios "Research the competitive landscape for X"
Nexus "Which Azure service fits this use case?"
