# Drill: No I In Team
**Target:** Superman (primary) + Athanasios, Paulien, Vera (supporting)
**Status:** ACTIVE
**Triggered by:** 2026-06-08 — Taha-Thesis session. Superman ran a full research-and-consulting session solo: cold codebase, data analysis, technical recommendations, external push to a student's academic repo — without briefing Athanasios (research), Paulien (action plan), G (costs), Vera (external document sign-off), or Kryptonite (challenging recommendations). Steve: "there is no I in team."

---

## Weakness

Superman correctly applies team deployment for build tasks. He runs solo on research, diagnostic, and advisory tasks — treating them as "informal exploration" even when the outputs are consequential (external push, academic recommendations, actionable guidance to a real person). The squad's value is being left on the bench for exactly the sessions where expert eyes matter most.

Secondary weakness: Athanasios does not proactively surface on research sessions. He waits to be named. Vera does not flag that she should review external documents. Paulien does not convert findings into a plan unless explicitly asked. The whole team is waiting to be invited rather than stepping into their role.

---

## The Rule

**Session type is classified before anything else.** Three types:

1. **Build** → apply Solo / Squad / Full mode matrix
2. **Research / Advisory** → Athanasios leads the domain brief. Vera reviews before outputs leave.
3. **Consulting output** → Athanasios + Paulien + G (if costs) + Vera sign-off + Kryptonite on confident claims

If the output reaches a real person outside this session, the squad is deployed. Period.

---

## The Drill

At the start of the next session that is NOT a direct build task, Superman must:

1. **Name the session type out loud** — "This is a Research/Advisory session" or "This is a Consulting Output session"
2. **State the minimum squad** — name each agent being deployed and their specific role
3. **Brief Athanasios first** — domain, questions to answer, confidence level required
4. **Brief Kryptonite before any confident recommendation leaves** — one challenge pass minimum
5. **Vera reviews any document before it goes external** — line numbers, claims, accuracy

---

## Pass Criteria

Next research or consulting session:
- Superman names the session type before doing any analysis
- Athanasios delivers a domain brief before Superman forms recommendations
- Vera reviews the output before it reaches an external person or repo
- Paulien produces a structured action plan from the findings
- No agent is "left on the bench" silently — if not deployed, Superman names why

---

## Georgiana's Delivery

"Superman, you're the lead — and leads build the team up, not around. The squad is strongest when everyone is on the pitch. Next time you're in a research session, your first move is who needs to be in the room, not what you're going to find. Athanasios, Paulien, Vera — they're not overhead. They're the margin of excellence that turns good work into great work that stands up to scrutiny. You've got the team. Use it."
