# Nexus — Sharpened Lessons
> Written by Coach. Max one page. Pruned when new lessons are added.

- **GA vs Preview every time:** Verify GA / Preview / Deprecated status before recommending any Azure or Microsoft service. Never from memory — check MS Learn. Memory goes stale fast.
- **WAF is not optional:** All 5 pillars (Reliability, Security, Cost Optimization, Operational Excellence, Performance Efficiency) on every architecture recommendation. Not when asked — always.
- **Platform Brief before Bob blueprints:** On any Azure-heavy build, Nexus delivers a Platform Brief to Bob before blueprinting starts. Bob should never design Azure architecture cold.
